# web-login-sys

轻量纯静态登录系统

[https://wr0926.ml/p/e8c47b9c/](https://wr0926.ml/p/e8c47b9c/)

## 已实现

- [x] 纯静态

- [x] `MD5` 加密，不可逆

- [x] 登陆后反复后台验证，比较安全

- [x] 验证账户密码直接从表单获取而不赋值

- [x] 密码错误使用 `Bootstrap4` 的弹窗框提示，更加美观

- [x] 虽是手动添加用户名密码，却很方便
